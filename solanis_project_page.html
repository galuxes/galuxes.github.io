Solanis Flight System

Project Overview

Solanis is a 3D flight-based exploration game where players navigate through stunning alien landscapes. As the lead programmer, I developed the core flight mechanics, implemented physics systems, and created design-focused tools for an immersive experience.

Flight System Implementation

1. Core Physics Framework

Implemented a custom physics system using Unreal Engine's built-in physics engine as a foundation. Created a modular architecture that allows for easy adjustment of flight characteristics.

  • Custom force calculations for lift, drag, and thrust
  • Variable physics timestep for smooth simulation
  • Collision detection and response system

2. Control System

Developed an intuitive control scheme that balances responsiveness with realistic flight behavior. Implemented both gamepad and keyboard/mouse support with customizable sensitivity.

  • Analog input processing for smooth control
  • Input mapping system for different control schemes
  • Control surface simulation for realistic movement

3. Camera System

Created a dynamic camera system that follows the player's craft while maintaining visibility and providing cinematic views. Implemented multiple camera modes for different gameplay situations.

  • Third-person chase camera with predictive movement
  • First-person cockpit view with motion effects
  • Cinematic camera modes for special moments

4. Environmental Interaction

Developed systems for the flight mechanics to interact with the environment, including wind currents, atmospheric effects, and terrain-based challenges.

  • Wind system affecting flight dynamics
  • Atmospheric density variations
  • Terrain-based turbulence and updrafts
